# Word (lemma + reading) as the learning unit

JKRT trains **reading of vocabulary in news**, not isolated kanji characters. Progress and review grades attach to a **Word** identified by **lemma + reading**; surface form is only contextual. Character-level `user_kanji` tracking was rejected because compounds like 経済 can be “known” as parts while still unreadable as a word — the failure mode that matters for N2→N1 news reading.

**Considered:** character-only progress (simpler schema, weaker product); word-level from day one (chosen); hybrid later if needed.
